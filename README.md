### OlaTagger

OlaTagger, written in pure Swift, helps you read and write MP3 ID3 tags easily.

* reading / writing MP3 ID3 tags version 2.3 and 2.4 (except for footer)
* compatible tags
  * TIT2 - title
  * TALB - album
  * TPE1 - artist
  * APIC - artwork
  * USLT - lyrics
